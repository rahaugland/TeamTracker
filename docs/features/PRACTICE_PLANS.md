# Practice Planning

Complete guide to building structured practice plans using the drill library in TeamTracker.

## Overview

Practice Plans help coaches organize training sessions with time-blocked schedules. Build plans using drills from your library, save templates for reuse, and link plans to practice events for automatic execution tracking.

## Key Concepts

### What is a Practice Plan?

A **practice plan** is a structured schedule for a training session, divided into time blocks.

**Components:**
- Plan name and metadata
- Time-blocked structure
- Drills from library
- Custom activities
- Total duration calculation
- Optional notes

### Practice Blocks

Each plan consists of multiple **blocks** representing segments of practice.

**Block types:**
- Warmup
- Drill
- Scrimmage
- Cooldown
- Break
- Custom

## Block Types

### Warmup

Preparation activities to start practice.

**Typical duration:** 10-15 minutes

**Common activities:**
- Dynamic stretching
- Light cardio
- Basic ball handling
- Team circle

**Tips:**
- Start every practice with warmup
- Increase intensity gradually
- Include volleyball-specific movements
- Set positive tone

### Drill

Skill-focused exercises from your drill library.

**Typical duration:** 10-30 minutes per drill

**Features:**
- Select from drill library
- Auto-fill duration from drill
- Link to drill details
- Track execution automatically

**Tips:**
- Use progressions
- Mix skill focuses
- Consider roster size
- Monitor intensity

### Scrimmage

Controlled or live game play.

**Typical duration:** 15-30 minutes

**Variations:**
- 6v6 full court
- Small-sided games
- Scoring variations
- Specific rotations

**Tips:**
- Include in every practice
- Apply skills from drills
- Adjust rules for focus
- Let players play

### Cooldown

End-of-practice recovery and review.

**Typical duration:** 5-10 minutes

**Common activities:**
- Static stretching
- Team huddle
- Practice review
- Announcements

**Tips:**
- Never skip cooldown
- Review key learnings
- Recognize effort
- Preview next practice

### Break

Rest period between activities.

**Typical duration:** 2-5 minutes

**Purpose:**
- Hydration
- Equipment setup changes
- Brief instruction
- Mental reset

**Tips:**
- Schedule strategic breaks
- Use for transitions
- Keep brief
- Players hydrate

### Custom

Free-form block for any activity.

**Use for:**
- Team meetings
- Video review
- Position-specific work
- Special activities

**Tips:**
- Use descriptive titles
- Add detailed notes
- Set appropriate duration
- Consider as template

## Creating Practice Plans

### Who Can Create Plans

- Head coaches
- Assistant coaches

### Basic Plan Creation

1. Navigate to **Practices**
2. Click **Create Practice Plan**
3. Enter plan name (e.g., "Tuesday Practice - Passing Focus")
4. Add blocks:
   - Click **Add Block**
   - Select block type
   - Set duration
   - Add details (select drill, enter custom title, etc.)
5. Reorder blocks by dragging
6. Review total duration
7. Click **Save**

### Step-by-Step Example

**Creating a 90-minute practice:**

1. **Warmup Block** (10 min)
   - Type: Warmup
   - Duration: 10
   - Note: "Dynamic stretching, partner pepper"

2. **Passing Drill** (20 min)
   - Type: Drill
   - Select: "Triangle Passing Drill" from library
   - Duration: 20 (auto-filled from drill)

3. **Setting Drill** (20 min)
   - Type: Drill
   - Select: "Setter Training - Quick Sets"
   - Duration: 20

4. **Break** (5 min)
   - Type: Break
   - Duration: 5

5. **Hitting Lines** (25 min)
   - Type: Drill
   - Select: "Hitting Lines with Defense"
   - Duration: 25

6. **Scrimmage** (20 min)
   - Type: Scrimmage
   - Duration: 20
   - Note: "Focus on transitioning from defense to offense"

7. **Cooldown** (10 min)
   - Type: Cooldown
   - Duration: 10

**Total:** 90 minutes

### Adding Drills to Blocks

When you select "Drill" block type:

1. Click **Select Drill**
2. Search or browse drill library
3. Click on drill to select
4. Duration auto-fills from drill default
5. Adjust duration if needed
6. Add block-specific notes

**Quick access:**
- Recently used drills shown first
- Filter by skill
- Filter by level
- Search by name

### Custom Block Details

For "Custom" blocks:

1. Enter descriptive title (e.g., "Position Meetings")
2. Set duration
3. Add notes with details
4. Save block

**Examples:**
- "Film Review - Last Game"
- "Setter/Libero Specific Work"
- "Mental Training - Visualization"
- "Team Building Activity"

## Managing Practice Plans

### Viewing Plans

1. Navigate to **Practices**
2. View list of all plans
3. Click plan to see details:
   - All blocks in order
   - Total duration
   - Creation date
   - Usage history

### Editing Plans

1. Open plan detail page
2. Click **Edit Practice Plan**
3. Modify:
   - Plan name
   - Add/remove blocks
   - Reorder blocks
   - Update durations
   - Change notes
4. Click **Save**

**Note:** Editing a plan doesn't affect past uses, only future.

### Duplicating Plans

Create copies for variations:

1. Open plan detail page
2. Click **Duplicate**
3. Plan name becomes "Copy of [Original]"
4. Edit the duplicate
5. Save as new plan

**Use cases:**
- Create variations for different levels
- Modify for roster size changes
- Adjust focus while keeping structure
- Build template library

### Deleting Plans

1. Open plan detail page
2. Click **Delete Practice Plan**
3. Confirm deletion

**Warning:** Deleting removes plan but preserves execution history.

**Recommendation:** Keep templates even if not currently using.

## Reordering Blocks

### Drag and Drop

On desktop:

1. Click and hold block
2. Drag to new position
3. Release to drop
4. Order updates automatically

### Move Up/Down Buttons

On mobile or alternative:

1. Click **Move Up** arrow
2. Or click **Move Down** arrow
3. Block shifts position
4. Repeat as needed

**Tips:**
- Logical flow: Warmup → Drills → Scrimmage → Cooldown
- Group similar skills
- Alternate intensity
- Strategic breaks

## Duration Management

### Block Durations

Set duration for each block in minutes.

**Recommendations:**

| Block Type | Typical Duration |
|-----------|------------------|
| Warmup | 10-15 min |
| Drill (simple) | 10-15 min |
| Drill (complex) | 20-30 min |
| Scrimmage | 15-30 min |
| Cooldown | 5-10 min |
| Break | 2-5 min |
| Custom | Varies |

### Total Duration

Automatically calculated:

- Sums all block durations
- Displayed at top of plan
- Updates in real-time
- Use to fit practice window

**Planning tips:**
- Account for setup time
- Add buffer for transitions
- Consider actual vs. planned time
- Adjust based on experience

### Time Management

**Best practices:**
- Plan 10% under available time (buffer)
- Track actual vs. planned duration
- Adjust future plans based on reality
- Don't over-pack practices

## Linking to Events

### Attaching Plans to Practices

Link plans to scheduled practice events:

**Method 1: During Event Creation**
1. Create practice event
2. In practice plan section, select plan
3. Save event

**Method 2: Edit Existing Event**
1. Open practice event
2. Click **Edit Event**
3. Select practice plan
4. Save

**Benefits:**
- Players see practice structure
- Automatic execution tracking
- Historical practice record
- Template effectiveness data

### Viewing Plans from Events

Players/parents can preview:

1. Open practice event
2. Click **View Practice Plan**
3. See block structure and drills
4. Understand practice focus

**Visibility:**
- Block types and durations visible
- Drill names visible
- Detailed descriptions hidden (coach-only)

## Execution and Tracking

### Automatic Execution Tracking

When practice event occurs with linked plan:

1. Event marked as completed
2. All drills in plan auto-marked as executed
3. Prompt for ratings on each drill
4. Execution history updated
5. Progression tracking activated

### Rating Practice

After practice, coaches rate each drill:

1. Open event detail
2. Navigate to practice plan section
3. Rate each drill (1-5 stars)
4. Add notes on execution
5. Save ratings

**Use ratings to:**
- Identify effective drills
- Determine progression readiness
- Adjust future plans
- Share with assistants

### Practice Notes

Add overall practice notes:

- What went well
- What needs improvement
- Player highlights
- Adjustments for next time

**Access:**
- Event detail page
- Practice plan history
- Analytics dashboard

## Best Practices

### Plan Creation

**Before creating:**
- Define practice goals
- Consider roster size
- Check equipment availability
- Review recent practices
- Note player progression levels

**During creation:**
- Logical flow
- Balanced skill coverage
- Appropriate intensity curve
- Realistic timing
- Include all essentials (warmup, cooldown)

**After creating:**
- Review total duration
- Check equipment needs
- Share with assistants
- Preview with second opinion

### Practice Structure

**Effective structure:**
1. Start with warmup (always)
2. Simple to complex drills
3. Individual → small group → team
4. Include competitive element
5. End with scrimmage/play
6. Finish with cooldown

**Intensity curve:**
- Gradual increase
- Peak mid-practice
- Taper toward end
- Never end on high-intensity

**Skill distribution:**
- Focus on 1-2 primary skills
- Touch all skills weekly
- Build on previous practices
- Progressive development

### Time Management

**Planning:**
- Use standard block durations
- Add transition buffers
- Plan 90% of available time
- Save time for unexpected

**During practice:**
- Start on time
- Stick to plan
- Adjust if needed
- End on time

**After practice:**
- Note actual durations
- Adjust templates
- Learn from timing
- Improve estimates

### Template Usage

**Build gradually:**
- Start with 3-4 templates
- Add as you refine
- Test before relying on
- Update based on feedback

**Maintain quality:**
- Review templates seasonally
- Remove ineffective structures
- Update drill selections
- Adjust durations

**Customize appropriately:**
- Don't blindly reuse
- Adjust for current needs
- Consider player development
- Account for weather/space

## Troubleshooting

### Can't Add Drill to Block

**Solutions:**
- Ensure drill exists in library
- Try refreshing drill library
- Check permissions
- Verify block type is "Drill"

### Total Duration Wrong

**Solutions:**
- Check all block durations
- Look for hidden blocks
- Refresh page
- Manually recalculate

### Plan Won't Save

**Solutions:**
- Ensure plan has name
- Check all required fields
- Verify internet connection
- Check sync status
- Try again

### Drill Not Tracking Execution

**Solutions:**
- Verify plan linked to event
- Check event occurred
- Ensure drill block (not custom)
- Try manual execution marking
