# TeamTracker Quick Start Guide

Get up and running with TeamTracker in 5 minutes.

## For Coaches

### Step 1: Create Your Account (30 seconds)

1. Open TeamTracker
2. Click **Sign in with Google**
3. Select **Head Coach** as your role

### Step 2: Create a Season (30 seconds)

1. Navigate to **Teams** page
2. Click **Add Season**
3. Enter:
   - Name: "Fall 2024" (or current season)
   - Start date and end date
4. Toggle **Active** on
5. Click **Save**

### Step 3: Create a Team (30 seconds)

1. Click **Add Team**
2. Enter team name (e.g., "Varsity Girls")
3. Select your season
4. Click **Save**

### Step 4: Add Players (2 minutes)

1. Navigate to **Players**
2. Click **Add Player** for each team member
3. Enter at minimum:
   - Name
   - Position(s)
4. Click **Save**
5. From each player's profile, click **Add to Team**
6. Select team and jersey number
7. Click **Save**

**Or import from Spond** (if you have existing data):
1. Navigate to **Import**
2. Upload your Spond CSV
3. Follow the wizard

### Step 5: Create Your First Event (1 minute)

1. Navigate to **Schedule**
2. Click **Add Event**
3. Fill in:
   - Type: Practice or Game
   - Title: "First Practice"
   - Date and time
   - Location
4. Click **Save**

### Step 6: Share Team Code (30 seconds)

1. Go to team detail page
2. Find **Team Invite Code**
3. Click **Copy Code**
4. Share with your players and parents via text/email

### You're Ready!

You can now:
- Track attendance at events
- Build practice plans with drills
- View analytics on attendance
- Manage your roster

**Next steps:**
- Build your drill library
- Create a practice plan
- Take attendance after your first event
- Explore analytics

## For Players

### Step 1: Create Account (30 seconds)

1. Open TeamTracker
2. Click **Sign in with Google**
3. Select **Player** as your role

### Step 2: Join Your Team (1 minute)

1. Get the team invite code from your coach
2. Navigate to **Join Team**
3. Enter the code
4. Click **Find Team**
5. Click **Join Team**

### Step 3: View Schedule (30 seconds)

1. Navigate to **Schedule** or **Dashboard**
2. See all upcoming events

### Step 4: RSVP to Events (30 seconds per event)

1. Click on an event
2. Click **Set RSVP**
3. Select **Attending**, **Not Attending**, or **Maybe**
4. Click **Save**

### You're Ready!

You can now:
- View your team's schedule
- RSVP to practices and games
- Track your attendance stats
- Update your profile

---

## For Parents

### Step 1: Create Account (30 seconds)

1. Open TeamTracker
2. Click **Sign in with Google**
3. Select **Parent** as your role

### Step 2: Link to Your Child (1 minute)

1. Navigate to **Dashboard**
2. Click **Link Player**
3. Search for your child's name
4. Select them from results
5. Click **Confirm**

### Step 3: RSVP for Events (30 seconds per event)

1. View your child's schedule on dashboard
2. Click on an event
3. Click **Set RSVP**
4. Select attendance status
5. Click **Save**

### You're Ready!

You can now:
- View your child's schedule
- RSVP on their behalf
- Monitor their attendance
- Link additional children if needed

## Essential First Steps

### Install the App

**Mobile:**
1. Open TeamTracker in your browser
2. Tap "Add to Home Screen" when prompted
3. Open from home screen

**Desktop:**
1. Look for install icon in browser address bar
2. Click "Install"
3. Launch from applications

### Enable Notifications (Optional)

1. Grant notification permissions when prompted
2. Stay updated on schedule changes
3. Receive reminders for upcoming events

### Complete Your Profile

1. Navigate to **Profile**
2. Add:
   - Phone number
   - Profile photo (URL)
3. Click **Save**

### Learn the Interface

**Navigation:**
- **Dashboard**: Overview and quick actions
- **Teams**: Manage teams and roster
- **Players**: View all players
- **Schedule**: Events calendar
- **Drills**: Practice drill library (coaches)
- **Practices**: Practice planning (coaches)
- **Analytics**: Attendance and stats (coaches)
- **Import**: Data import (coaches)

**Sync Indicator:**
- Shows online/offline status
- Displays pending changes count
- Click to force sync

## Common First Tasks

### As a Coach

**Take attendance at first practice:**
1. Go to event detail page
2. Click **Mark Attendance**
3. Mark each player (Present/Absent/Late/Excused)
4. Click **Save**

**Create first practice plan:**
1. Navigate to **Drills**
2. Add a few basic drills
3. Go to **Practices**
4. Click **Create Practice Plan**
5. Add drill blocks
6. Save plan

**View team analytics:**
1. Navigate to **Dashboard** or **Analytics**
2. Select your team
3. Choose date range
4. Review attendance patterns

### As a Player

**Update your profile:**
1. Go to **Profile**
2. Add phone number
3. Add profile photo URL
4. Click **Save**

**RSVP to next week's events:**
1. View **Schedule**
2. For each upcoming event:
   - Open event
   - Set RSVP
   - Save

### As a Parent

**Check upcoming schedule:**
1. Open **Dashboard**
2. View next 7 days of events
3. Note dates and times

**RSVP for the week:**
1. Click on each event
2. Set attendance status
3. Save each one

## Tips for Success

### For Coaches

- **Import historical data** from Spond to get started quickly
- **Create seasons** before teams
- **Use team invite codes** instead of manually adding users
- **Build drill library** gradually as you plan practices
- **Mark attendance** immediately after events while fresh

### For Players

- **RSVP early** so coach can plan
- **Update RSVPs** if plans change
- **Check schedule** at least weekly
- **Install the app** for quick access

### For Parents

- **Link all children** at once
- **Coordinate with child** before RSVPing
- **Set reminders** for regular RSVP check-ins
- **Add to calendar** by noting important dates

## Troubleshooting Quick Fixes

### "No active season" error
→ Create a season and toggle it active

### Can't find team to join
→ Get new code from coach, check for typos

### Changes not saving
→ Check sync status, ensure you clicked Save

### Can't see drills/practices
→ These features are coach-only

### Import failing
→ Ensure CSV format, map required columns
